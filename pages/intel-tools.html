Page Contract

Every future Intel Tools page should preserve this structure, then swap in the country or region source list from the resource guide.

Source Cards

Each tool needs a clear operational use case, active link, concise visible summary, and hover/focus overlay for deeper context.

Visual Context

Pages must include maps, flow diagrams, matrices, tables, website previews, or code examples instead of becoming text-only directories.

Analyst Workflow

Sections should map tools to questions: what happened, where, who moved, what changed, and which source confirms it.

Validation

Internal links, external links, overlay clipping, responsive layout, and settings-theme behavior should be checked before rollout.
