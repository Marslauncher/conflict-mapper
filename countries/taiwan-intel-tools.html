Code Examples

Examples are intentionally practical and credential-safe. Secrets remain placeholders and should be passed from environment variables in production automation.

Sentinel-2 Taiwan Strait

Python 
# pip install sentinelhub
import os
from sentinelhub import (
    BBox,
    CRS,
    DataCollection,
    MimeType,
    SentinelHubRequest,
    SHConfig,
    bbox_to_dimensions,
)

config = SHConfig()
config.sh_client_id = os.environ["CDSE_CLIENT_ID"]
config.sh_client_secret = os.environ["CDSE_CLIENT_SECRET"]

# Bounding box: Taiwan Strait region
bbox = BBox(bbox=[119.5, 24.5, 122.5, 26.5], crs=CRS.WGS84)
size = bbox_to_dimensions(bbox, resolution=60)

evalscript = """
//VERSION=3
function setup() {
  return { input: ["B04", "B03", "B02"], output: { bands: 3 } };
}
function evaluatePixel(sample) {
  return [sample.B04 / 3000, sample.B03 / 3000, sample.B02 / 3000];
}
"""

request = SentinelHubRequest(
    evalscript=evalscript,
    input_data=[SentinelHubRequest.input_data(
        data_collection=DataCollection.SENTINEL2_L2A,
        time_interval=("2026-05-01", "2026-06-07"),
        mosaicking_order="leastCC",
    )],
    responses=[SentinelHubRequest.output_response("default", MimeType.PNG)],
    bbox=bbox,
    size=size,
    config=config,
)

image = request.get_data()

ADS-B Near Taiwan

Python 
import requests

# Aircraft within 250 nautical miles of northern Taiwan.
url = "https://api.adsb.lol/v2/point/25.0/121.5/250"
response = requests.get(url, timeout=30)
response.raise_for_status()
data = response.json()

for ac in data.get("ac", [])[:10]:
    flight = (ac.get("flight") or "?").strip()
    print(
        f"ICAO: {ac.get('hex')} | "
        f"Flight: {flight} | "
        f"Type: {ac.get('t', '?')} | "
        f"Alt: {ac.get('alt_baro', '?')} ft | "
        f"Military: {ac.get('military', '?')}"
    )

OSM Infrastructure Query

Python 
import requests

overpass_url = "https://overpass-api.de/api/interpreter"
query = """
[out:json][timeout:60];
(
  node["aeroway"="aerodrome"](21.8,119.0,25.5,122.4);
  way["aeroway"="aerodrome"](21.8,119.0,25.5,122.4);
  node["harbour"](21.8,119.0,25.5,122.4);
  way["harbour"](21.8,119.0,25.5,122.4);
);
out body;
>;
out skel qt;
"""

response = requests.post(overpass_url, data={"data": query}, timeout=90)
response.raise_for_status()
data = response.json()

for element in data.get("elements", [])[:10]:
    print(element["type"], element["id"], element.get("tags", {}))

Template Validation Notes

This section is included so future country pages preserve the same implementation contract instead of becoming unstructured link dumps.

What Must Stay

  • Every source card has an active link and a hover/focus overlay.
  • Visible copy stays concise, with deeper context in overlays.
  • The page contains images, preview modules, tables, and code blocks.
  • External examples do not hardcode secrets or credentials.

Rollout Rule

When this template is approved, country pages should be generated one country at a time from the resource guide. Each rollout should validate external link reachability, internal nav links, overlay clipping, mobile layout, and theme/text-size propagation before the next country is added.
