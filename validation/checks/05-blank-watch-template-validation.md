# Blank Watch Template Validation

Use this before using the blank watch-page template for any new watch page.

Do not make code changes during this validation pass. Produce findings only.

## Check

- Template contains no country/theater-specific analysis content.
- Template includes section titles and empty/skeleton content cards only.
- Required watch-page sections exist:
  - Strategic Brief
  - Current Regional Assessment
  - Recent Think Tank Coverage
  - Operational Map
  - Situational Status
  - Weather & Sea State
  - Intelligence Feed
  - Force Comparison
  - Current Assessment
  - Things To Note
  - Things To Watch
  - Escalation Likelihood
- Layout works on desktop and mobile.
- Cards visually match the approved watch-page style.
- Placeholder/skeleton content does not look like published analysis.
- Template can be copied for a future watch page without inheriting Taiwan/Korea-specific feed filters or text.

## Output

- Pass/fail checklist.
- Screenshot paths.
- Any template content that should be removed before reuse.
